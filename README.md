## Installation

1. Clone the repo:
   ```bash
   git clone https://github.com/Sainvi-j/workspace-ui
   ```
````

2. Install dependencies:
   ```bash
   npm install
   ```
3. Run locally:
   ```bash
   npm run dev
   ```
4. Open `yourlocalhostsite` in your browser.
